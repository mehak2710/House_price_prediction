
# 🏠 House Price Prediction App

A beginner-friendly and realistic **House Price Prediction** web application built using **Streamlit**.
The app trains a machine learning model on the dataset during runtime and delivers fast, accurate predictions through a clean and interactive interface.

## 🚀 Tech Stack

* **Python**
* **Streamlit** – Web UI
* **Pandas** – Data Processing
* **Scikit-Learn** – Machine Learning (Random Forest Regressor)

## ⭐ Features

* 📊 Trains an ML model instantly when the app loads
* 🧮 Predicts house prices using essential features:

  * Area (sq ft)
  * Bedrooms
  * Bathrooms
  * Location
* ⚡ Fast prediction workflow
* 🎨 Minimal and user-friendly Streamlit interface
* 🔧 Easily extendable with more real estate attributes

## 📈 Key Performance Indicators (KPIs)

1. **Model Training Time**
   Measures how quickly the model trains on the dataset when the app starts.

2. **Prediction Speed**
   Evaluates how fast the model generates a price estimate.

3. **Feature Impact Insight**
   Shows how different variables (area, bedrooms, location, etc.) influence house prices.

4. **Data Consistency Check**
   Ensures smooth alignment between dataset columns and user input for accurate predictions.
